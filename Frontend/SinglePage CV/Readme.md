# SinglePage CV

## 项目简介

这是一个单页简历学习实践网页，展示了个人信息、技能、教育背景、工作经验以及互联网链接。

## 使用的技术

### HTML

- 使用语义化标签（如 `<header>`、`<section>`、`<footer>`）构建页面结构。
- 使用 `<h1>` 至 `<h3>` 标签定义标题层次。
- 使用 `<address>` 标签展示联系信息。
- 使用 `<ul>` 和 `<li>` 标签列出成就和技能。

### CSS

- 使用选择器和属性控制页面样式。
- 设置全局字体、行高、颜色等基础样式。
- 使用 `margin` 和 `padding` 控制布局和间距。
- 使用 `border` 和 `width` 定义分割线样式。
- 使用 `text-align` 和 `font-style` 控制文本对齐和样式。
- 响应式设计：通过百分比宽度（如 `width: 90%` 和 `padding: 0 5%`）实现页面适配不同屏幕。
