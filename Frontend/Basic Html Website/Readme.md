# Basic Html Website

## 项目简介
这是一个基础的前端开发布局学习项目，采用了 HTML 和 CSS 技术构建，实现了网格布局设计。

## 技术特点

### 布局结构
- 使用 CSS Grid 布局实现整体页面结构
- 采用 grid-template-areas 定义网格区域
- 页面分为头部导航、左侧栏、主内容区、右侧栏和页脚五个主要部分

### HTML 特性
- 使用语义化标签（header、nav、main、aside、footer）
- 清晰的内容结构组织
- 使用 section 和 article 标签划分内容区块
- 响应式图片和图标的使用

### CSS 功能
- Flexbox 导航栏布局
- Grid 系统用于页面整体布局
- 响应式设计
- Google Fonts 字体引入
- 统一的样式主题
- 卡片式设计（review 部分）

## 项目结构

```
Basic Html Website/
├── BasicHtmlWebsite.html    # 主要的 HTML 文件
├── BasicHtmlWebsite.css     # 样式表文件
└── icon.png                 # 导航栏图标
```

## 页面组成部分
1. 导航栏（包含 logo、名称和导航链接）
2. 页面头部（职位和简介）
3. 左侧项目列表
4. 主要内容区（工作经验和教育背景）
5. 右侧评价区
6. 页脚信息

## 布局特点
- 使用 1:2:1 的三栏布局比例
- 响应式间距管理
- 清晰的视觉层次
- 一致的边框和间距处理
